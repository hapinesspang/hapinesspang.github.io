ClipQueue 지원

ClipQueue는 여러 문장을 큐에 저장하고 키보드에서 순서대로 입력하는 iOS 앱입니다.

자주 묻는 질문

다른 앱에서 복사한 내용을 자동으로 계속 저장하나요?
아니요. iOS 제한과 개인정보 보호를 위해 사용자가 저장 버튼 또는 단축어 액션을 실행할 때만 현재 클립보드를 저장합니다.

키보드에서 광고가 나오나요?
아니요. 광고는 메인 앱 화면에서만 표시될 수 있습니다.

키보드 전체 접근은 왜 필요한가요?
저장한 큐와 문구를 키보드에서 읽고 사용 완료 상태를 갱신하기 위해 필요합니다. ClipQueue 키보드는 사용자가 타이핑하는 내용을 수집하거나 서버로 보내지 않습니다.

저장한 문구가 서버로 전송되나요?
아니요. 현재 버전은 자체 서버를 사용하지 않고 기기 내 저장소를 사용합니다.
